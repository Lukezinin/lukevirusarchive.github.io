           COLLABVM VIRUS: PROGRESSIVE APOCALIPSE           
============================================================

DESCRIPTION:
This project is a progressive malware simulation designed 
to demonstrate visual GDI manipulation and system 
interaction within controlled environments. 

The program evolves through several stages of intensity, 
ending with an automated system lockdown.

------------------------------------------------------------
[!] WARNING:
This software is intended for educational purposes and 
virtual machine testing ONLY. Running this on a host 
machine will result in permanent data loss and boot 
failure. Use strictly within isolated environments.
------------------------------------------------------------

TECHNICAL STAGES:
The chaos is divided into time-based increments:

* Phase 1 (0s - 2s): Screen oscillation and vertical 
  pixel displacement.
* Phase 2 (2s - 4s): Linear tunneling and color 
  inversion trailing.
* Phase 3 (4s - 6s): High-frequency neon pulsing via 
  bitwise logic.
* Phase 4 (6s - 8s): Kaleidoscope mirroring and 
  recursive quadrant shifting.
* Final Phase (8s+): Infinite scaling and bit-shredding.

EXECUTION PROFILE:
* MBR Interaction: Overwrites the Master Boot Record 
  on PhysicalDrive0.
* Data Removal: Recursive background deletion of 
  user directory contents.
* Direct Execution: No confirmation dialogs or 
  user interface. Activation is immediate.

USAGE:
The binary is designed for the Windows environment. 
Once executed, it will cycle through all phases 
automatically without requiring user input.
